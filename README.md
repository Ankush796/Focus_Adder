# Telegram Adder Manager Bot

Dashboard-style Telegram bot with:
- OTP login (StringSession)
- MongoDB backed storage
- Hourly member add limits
- Duplicate prevention
- Background worker

## Setup
1. Create `.env`
2. Install requirements
3. Run `python main.py`

## Notes
- Bot is UI + control panel
- User sessions are stored securely
- Limits are enforced per account

⚠️ Use responsibly.
